QuickWatch | Full-Stack Movie Ticketing Platform


Developed a scalable movie booking web application using the MERN stack (MongoDB, Express, React, Node.js) and integrated TMDB API for real-time movie metadata.
Implemented secure, multi-method authentication (Email, Social, Phone) and multi-session management using Clerk.
Engineered a custom seat reservation engine that holds selected seats for 10 minutes during the payment window, automatically releasing them upon timeout or failure .
Automated transactional workflows, including booking confirmations and showtime reminders, by scheduling background jobs with Inngest.
Designed an Admin Dashboard to manage movie listings, show timings, and system activity.

Deployment Link: https://quickwatch-khaki.vercel.app/
